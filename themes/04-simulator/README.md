# テーマ04: 可逆 Turing 機械シミュレータと可視化

## 問題設定

可逆 TM の「実行を逆再生すると逆関数が計算される」（Lecerf 反転）を、
目で見て分かる形にする。とっかかりとして**動く最小実装**を同梱した。

```bash
python3 rtm_sim.py
```

Morita 流の四つ組（quadruple）形式の可逆 TM を前進実行し、機械的に
構成した逆機械 reverse(M) が実行トレースをそのまま逆向きに再生して
入力を復元する様子を ASCII で表示する。

## 同梱実装の中身

- 四つ組形式: 書換規則 (q, s → s', q') とシフト規則 (q, ±1, q')
- `reverse(M)`: 規則を字義どおり逆転（書換は s ↔ s'、シフトは符号反転）
- デモ機械 FLIP: ビット列を反転して停止する可逆 TM
- 検証: 前進トレースと逆機械のトレースが鏡像になることを assert

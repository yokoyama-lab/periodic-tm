# 課題3: 前周期階層の正確な境界 — 特徴づけ定理（証明済み）

f^{m+p} = f^m を満たす計算可能な f について、分解 f = ι₁ ∘ ι₂ ∘ e
（ι は計算可能な対合、e は計算可能な冪等）がいつ可能かの正確な条件。
既存の no-go（前周期 2 で不可能な例の存在、Lean 証明済み）を
「なぜその例で壊れるか」まで特定して特徴づけに昇格させる。

## 準備

最終像（periodic core）を R := Im(f^m) とおく。f^{m+p} = f^m より
f(R) ⊆ R かつ f^p|_R = id なので f|_R は R 上の位数 p の全単射である。
各 x は m ステップ以内に R に落ちる: ρ(x) := min { j ≤ m : f^j(x) ∈ R }。

## 観察 3.0（三因子形は前周期 1 を構造的に強制する）— 証明済み

Im(e) = R の下で文字どおりの f = ι₁ ∘ ι₂ ∘ e（ι は R 外で恒等に
正規化）を要求すると、右辺の像は R に含まれるから Im(f) ⊆ R、
すなわち前周期 m ≤ 1 が**計算可能性と無関係に**強制される。
前周期 2 以上で問うべき正しい形は、次の「核分解」である。

## 定理 3.1（核分解の特徴づけ）— 証明済み

f の**核分解**とは、計算可能な冪等 e（Im(e) = R, e|_R = id、
すなわち核への射影）と計算可能な対合 ι₁, ι₂ の組で

  f ∘ e = ι₁ ∘ ι₂ ∘ e（したがって R 上で f = ι₁ ∘ ι₂）

を満たすもの。前周期 1 では f∘e = f なのでリポジトリの
`index_one_decomp` の形に一致する。

**定理.** 次は同値。
(i) R が決定可能。
(ii) f の核分解が存在する。

**証明 (i)⇒(ii).** R が決定可能なら ρ(x) は計算可能（j = 0..m を順に
検査）。e(x) := f^{ρ(x)}(x) とおくと e は計算可能、Im(e) = R、
x ∈ R なら ρ(x) = 0 で e(x) = x、ゆえに e∘e = e（冪等）。
f|_R は位数 p の計算可能な全単射なので軌道はすべて有限（長さは p の
約数）であり、局所有限分解定理（research/04, 定理 4.1）により
R 上で f|_R = ι₁ ∘ ι₂（計算可能な対合）。R の決定可能性を使って
ι_i を R の外では恒等写像として全域に拡張する（対合性は保たれる）。
すると任意の x で ι₁(ι₂(e(x))) = f(e(x)) = f(f^{ρ(x)}(x)) = f^{ρ(x)}(f(x))。
e(x) ∈ R かつ R 上で ι₁∘ι₂ = f だから ι₁(ι₂(e(x))) = f(e(x))、
すなわち ι₁∘ι₂∘e = f∘e。これが核分解である。∎

（執筆時の注記: 当初は文字どおりの f = ι₁∘ι₂∘e を主張していたが、
証明を書く過程で観察 3.0 の構造的障害に気づき、主張を核分解に
修正した。この「三因子形は m ≤ 1 でしか意味を持たない」こと自体が
副産物の発見である。）

**証明 (ii)⇒(i).** e が計算可能な冪等なら Fix(e) = { x : e(x) = x } は
決定可能（e(x) = x を検査するだけ）。冪等性より Im(e) = Fix(e)。
仮定 Im(e) = R より R は決定可能。∎

## 系 3.2（no-go の正体）

Lean 証明済みの前周期 2 no-go 例は「R = Im(f²) が決定不能になるよう
停止問題の尾を編み込んだ f」である。定理 3.1 はこれが本質であることを
示す: **分解を妨げるものは最終像の決定不能性、それだけである。**
前周期の深さ 2 という数自体に魔法はない — 前周期 1 で分解が常に
可能なのは、f を1回通すだけで強制的に R に入るため R の決定が
不要になるからである。

## 系 3.3（算術的階層での位置）

R = Im(f^m) は Σ⁰₁。よって「射影形分解が存在するか」（インデックスが
与えられた f に対する判定）は「Σ⁰₁ 集合が決定可能か」という形の問題で、
Σ⁰₃ に入る。下界（Σ⁰₃ 完全性）の証明は未着手 — Rice 型の定理と
「computable = Δ⁰₁」判定の既知の完全性結果（Σ⁰₃ 完全）を移植すれば
落ちる見込みが高い。次の一歩として残す。

## 残課題（正直な記録）

- 定理 3.1 は e を「R への射影」に正規化した形。一般の
  f = ι₁∘ι₂∘e（Im(e) が R と異なることを許す）で同じ特徴づけが
  成り立つかは未解決。(ii)⇒(i) の議論は Fix(e) の決定可能性しか
  与えず、Fix(e) ⊋ R の分解が存在する可能性を排除できていない。
  小さい有限モデルでの反例探索（全探索）が有効なはず。
- 系 3.3 の Σ⁰₃ 完全性の証明。
- 定理 3.1 の Lean 機械化（`PrePeriod.lean` の隣に置く。定理 4.1 の
  機械化が前提になる）。

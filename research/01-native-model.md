# 課題1: 位数 n ネイティブ機械モデル — 未解決。候補3案の失敗の解剖

目標（再掲）: 機械クラス C_n で (S) 健全性: 計算する関数は fⁿ = id、
(C) 完全性: fⁿ = id な計算可能関数は全部入る、(M) 所属決定可能、
の3条件を満たすもの。n = 2 は ITM。n ≥ 3 は未解決。

今回試した3案と、それぞれがどこで死ぬかの記録。

## 案A: 巡回トラック機械（wreath 積型）

テープを n 本のトラックに分け、機械を「トラック巡回シフト σ ∘ 各トラック
共通の変換 D」の形 F = σ ∘ D に制限する。σⁿ = id なので位数 n が
出そうに見える。

**失敗の解剖.** Fⁿ = (σD)ⁿ = (σⁿ)(σ^{-(n-1)}Dσ^{n-1})⋯(σ^{-1}Dσ)(D)。
これは「D の n 個の共役の積」であり、id になるのは D がその共役たちと
特殊な関係を満たすときだけ。D に条件を課すと今度はその条件自体が
「積が id」という**意味論的**条件になり、(M) が壊れる。σ と可換な D に
制限すれば Fⁿ = Dⁿ となり、問題が「Dⁿ = id なる D」に**そのまま退行**する
（何も解いていない）。教訓: 群論的に位数 n を外付けする構成は、
位数条件を内側の部品に押し付け直すだけで、真の削減にならない。

## 案B: 時間反転の n 乗根（物理アナロジー）

n = 2 の急所は「実行の逆再生」という機械操作が位数 2 であること。
では位数 n の機械操作は何か。候補として「実行トレースの巡回的
リインデックス」（計算履歴 c₀ → c₁ → ⋯ → c_T を位相 θ ∈ Z/n で
ずらして再開する）を検討した。

**失敗の解剖.** 逆再生が機械操作たりえるのは、可逆機械では
**局所遷移規則の字義的逆転**がそれを実現するからである（Lecerf）。
巡回リインデックスに対応する局所規則変換を探すと、「トレースの
1/n 地点」という**大域的な情報**（総ステップ数 T）が必要になり、
局所規則では実現できない。T を状態に持たせると停止性・可逆性が
壊れる。教訓: Z/2 の特権性は「逆転が局所的」という一点にあり、
n ≥ 3 の位相シフトは本質的に非局所。この障害は避けられない疑いが
強く、**「(S)+(C)+(M) を満たす C_n は n ≥ 3 で存在しない」という
不可能性定理の方が真かもしれない**。攻めるならこちら:
(M) を満たす任意の構文クラスが (S) を持つなら (C) が破れることを、
構文クラスの列挙可能性 + 対角線論法で示せないか。次の一手として最有力。

## 案C: 検証付きペア（証書モデル）

機械 = (M, 証書)。証書は「Mⁿ = id の証明」ないし「二対合分解
M = I₁∘I₂ の明示」。(M) は証書検査で決定可能、(C) は分解定理から、
(S) は証書の健全性から成立 — 3条件が全部立つ。

**失敗の解剖（なぜこれで解決と言えないか）.** これは ITM の
「機械の対称性が性質を**生む**」という美点を放棄している。ITM は
証明を運ばない — 遷移表の対称性という**検査可能な形**が involution 性と
一致する、そこが定理だった。証書モデルでは「構文 = 意味」が
「構文 = 意味の証明の添付」に弱まり、n = 2 との対比で得るものがない。
ただしこの案は基準として有用: 探すべきは「証書が機械の形そのものに
溶けている」モデルであり、証書モデルとの差分（証書の自明化可能性）が
問題の難しさの正確な定式化を与える。

## 現在地と次の一手（優先順）

1. **不可能性の方向を攻める**（案Bの教訓）: 「n ≥ 3 で (S)(C)(M) を
   同時に満たすクラスは存在しない」の証明を試みる。骨子案:
   (M) よりクラスは決定可能な機械集合、(S) より全員 fⁿ = id を計算、
   (C) が成り立つと仮定して、クラスの列挙から位数 n 関数の効果的
   列挙を作り、対角線で位数 n の計算可能関数でクラス外のものを構成
   できないか。障害: 対角化で作る関数の位数を n に制御する部分が
   非自明（位数は大域的性質で、対角化の局所的変更と相性が悪い）。
2. 不可能性が落ちない場合、(M) を半決定可能に緩めた変種で存在を探す。
3. n = 3, ドメイン有限（|X| ≤ 6 程度）の全探索で、(S)(C) を満たす
   最小の「規則対称性」を機械的に探す実験。パターンが見えれば
   定義の候補になる。

**結論: 未解決。ただし「解けない方向の証明」への転回という
明確な次の一手を得た。** これ自体が本課題の今回の主な進展である。

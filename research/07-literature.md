# Phase 0 の結果: 文献調査と、それが即座に生んだ部分定理

2並列の文献調査（Semantic Scholar/arXiv/Springer、書誌は実在確認済み）の
結論と、その情報から直ちに導けた新しい補題・定理の記録。

## 調査結論: 問いは未解決（文献にない）

「計算可能な対合 ι で ι f ι = f^{-1} を持たない計算可能置換は存在するか」
（= 計算可能置換の群 G は strongly real か）を直接扱う文献は
**見つからなかった**。これは研究テーマとして空いている。周辺の確定事項:

- **Kent (1962)**, *Constructive analogues of the group of permutations of
  the natural numbers*, Trans. AMS 104, 347–362。定理 1.7: G において
  サイクルタイプが共役の完全不変量になるのは**無限軌道が有限本**のとき
  ちょうど。定理 1.3: 軌道が creative 集合（K など）そのものになる
  計算可能置換の構成。
- **Higman (1990)**, *Transversals and conjugacy in the group of recursive
  permutations*, Groups—Canberra 1989, Springer LNM 1456, 142–160。
  定理 3.1: **すべての横断集合（軌道の代表系）が immune** になる、
  無限本の Z 鎖から成る計算可能置換の存在。← 反例の最有力素材。
- **Boege (arXiv:1612.05136)**, *On permutations with decidable cycles*。
  軌道同値関係が決定可能なクラス Perm では有効共役が完全に制御できる
  （定理 3.8）。r.e. 横断集合の存在 ≡ 軌道決定可能性（定理 4.4）。
  反例はどれも「無限本の無限鎖」を持たねばならない（命題 4.2）。
- O'Farrell–Short 系（reversibility 学派）は「2対合の積 = strongly
  reversible」の語彙と古典構成を持つが、計算可能設定は未着手。
- ceer 側（Andrews–Sorbi）は次数構造の研究で、個別 ceer の計算可能
  自己同型 vs 逆置換は扱っていない。05 計画の ceer 翻訳は近道に
  ならないと判明 → **主戦場は計算可能置換群の言葉で進める**。

## 調査結果から直ちに得られた新結果

### 補題 7.1（中心抽出）— 証明済み

f を計算可能置換、ι を計算可能な反転対称性（ι f ι = f^{-1}、ι² = id）
とする。ι が Z 鎖 C を（集合として）固定するなら、C の「中心」
（ι の固定点、または ι(x) = f(x) なる隣接対の x）が存在し、C の
任意の元から**計算可能に**見つかる。

**証明.** ι|_C は Z の向き反転同型、すなわち反射 k ↦ c − k。
c が偶数なら固定点 1 個、奇数なら ι(x) = f(x) なる x が 1 個。
x ∈ C から両方向へ k = 0, 1, 2, … と歩き、f^±k(x) で
ι(y) = y または ι(y) = f(y) を検査する。中心は x から有限距離に
あるから探索は停止する。∎

### 定理 7.2（Higman 置換の反転対称性は鎖をほぼ固定できない）— 証明済み

f を Higman の置換（全横断集合が immune）とする。f の計算可能な
反転対称性 ι は、**高々有限本の鎖しか固定できない**（無限本の鎖は
すべて別の鎖と対で交換される）。

**証明.** ι が無限本の鎖を固定すると仮定する。固定される鎖を列挙
しながら補題 7.1 で各中心を計算すると、中心の集合は無限 c.e. 集合で、
相異なる鎖から 1 点ずつ取るから、ある横断集合の部分集合に拡張できる。
これは全横断集合の immune 性に矛盾。∎

### 定理 7.3（正方向: 軌道決定可能なら 2 対合の積）— 証明済み

f の軌道同値関係が決定可能（f ∈ Perm、特に r.e. 横断集合 T を持つ:
Boege 定理 4.4）なら、f は 2 つの計算可能な対合の積である。

**証明.** x が与えられたとき、T の列挙と x の軌道の両方向の走査を
ドーブテイルすると、x の鎖の代表 t ∈ T が有限時間で見つかる
（代表は存在し、走査は各元を有限時間で訪れる）。t を原点として
鎖ごとの反射 ι₂(f^k t) = f^{−k} t, ι₁ = f∘ι₂ を定めればよい。
有限軌道は最小元を原点にする（定理 4.1 の構成）。∎

これは定理 4.1（局所有限）と補題 4.3（単一鎖）を同時に一般化する
決定版の正方向である。

## 残る唯一のギャップと、更新された攻略計画

定理 7.2 により、Higman 置換に対する反例証明の残りは:

> **残ギャップ.** ほぼすべての鎖を**対で交換する**計算可能な対合 ι
> （ι f ι = f^{-1}）の可能性を排除する。

交換対 {C, ι(C)} には中心が存在しないため補題 7.1 が使えない。
ただし ι は C と 逆向きの ι(C) の計算可能な整列（アラインメント）を
与えるので、「対ごとの代表選択」は依然として横断集合の**半分**を
選ぶことに相当する。攻め筋（優先順）:

1. **immunity の強化で潰す**: Higman の構成を改造し、「鎖の対 +
   計算可能アラインメント」からも無限 c.e. 部分横断集合が出るように
   する（対 {C, C'} とアラインメント ι から C∪C' 内の正準有限集合
   — 例えば ι(x) と x の「衝突点」— を抽出できないか。C と C' は
   別の鎖なので衝突しない。素朴には出ない。要工夫）。
2. **構成し直す**: chain_builder.py の枠組みで、Higman の要件
   （横断集合の immune 化）と 06 ノートの場合C排除（対交換の確約
   破壊）を同時に満たす優先度構成を直接書く。06 の「観測済み出力を
   避ける競走」がここで効く。
3. **弱い結果で確定させる**: 「G は strongly real でない **または**
   Higman 置換の反転対合はすべて有限固定」— 定理 7.2 自体が既に
   発表価値のある新観察なので、ギャップが埋まらなくても Kent–Higman–
   Boege の系譜に足す短報として成立する。

## 補足（本プロジェクトへの位置づけ）

- 定理 7.3 は「ITM が基底をなす範囲」の決定版: **軌道が効果的に
  見渡せる計算可能置換はすべて可逆分解できる**。境界は正確に
  「軌道決定可能性」にある（Perm の外に反例候補、中でも Higman 置換）。
- 05 計画の Phase 0 は完了。Phase 2 の単一撃破ゲーム（06）は
  上記攻め筋 2 の部品としてそのまま生きる。
